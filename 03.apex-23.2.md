## Oracle APEX 23.2 install on Oracle Linux 8.9:





### Architecture: 

```
Oracle Linux 8.9
   ↓
Oracle Database (PDB)
   ↓
APEX 23.2 or 24.2 (inside DB)
   ↓
ORDS (Java)
   ↓
Browser
```


### Prerequisites:
- Oracle Linux Server 8.9
- Oracle Database 19c or 21c installed and running
- At least 2GB RAM, 10GB disk space
- Root or sudo access
- Required components
    - SQL*Plus or SQLcl
    - Java-11 or 17 (for ORDS later)

```
vim $ORDS_CONFIG/global/settings.xml

<?xml version="1.0" encoding="UTF-8"?>
```

### HAProxy Configuration for ORDS/APEX:

```conf
# cat /etc/haproxy/haproxy.cfg

#---------------------------------------------------------------------
# Global settings
#---------------------------------------------------------------------
global
    log         127.0.0.1 local2

    chroot      /var/lib/haproxy
    pidfile     /var/run/haproxy.pid
    maxconn     4000
    user        haproxy
    group       haproxy
    daemon

    # turn on stats unix socket
    stats socket /var/lib/haproxy/stats

    # utilize system-wide crypto-policies
    ssl-default-bind-ciphers PROFILE=SYSTEM
    ssl-default-server-ciphers PROFILE=SYSTEM

#---------------------------------------------------------------------
# common defaults that all the 'listen' and 'backend' sections will
# use if not designated in their block
#---------------------------------------------------------------------
defaults
    mode                    http
    log                     global
    option                  httplog
    option                  dontlognull
    option http-server-close
    option forwardfor       except 127.0.0.0/8
    option                  redispatch
    retries                 3
    timeout http-request    10s
    timeout queue           1m
    timeout connect         10s
    timeout client          1m
    timeout server          1m
    timeout http-keep-alive 10s
    timeout check           10s
    maxconn                 3000


# Frontend - HTTP (redirect to HTTPS)
#---------------------------------------------------------------------
frontend ft_http
    bind *:80
    mode http
    option httplog

    ## Redirect all HTTP to HTTPS
    #redirect scheme https code 301 if !{ ssl_fc }



# Frontend - HTTPS
#---------------------------------------------------------------------
frontend ft_https
    bind *:80
#    bind *:443 ssl crt /etc/haproxy/certs/apex.pem 
    mode http
    option httplog

    ## Forward correct protocol
    #http-request set-header X-Forwarded-Proto https

    # ACLs for routing
    acl is_apex      path_beg /ords
    acl is_images    path_beg /i/

    # Route traffic
    use_backend be_apex_ords    if is_apex
    use_backend be_images  if is_images

    default_backend be_apex_ords


# Backend - ORDS/Tomcat
#---------------------------------------------------------------------
backend be_apex_ords
    mode http
    balance roundrobin

    #option forwardfor
    #http-request set-header X-Forwarded-Proto https

    server ords1 127.0.0.1:8080 check


# Backend - APEX Static Images
#---------------------------------------------------------------------
backend be_images
    mode http
    balance roundrobin

    server ords1 127.0.0.1:8080 check

```
