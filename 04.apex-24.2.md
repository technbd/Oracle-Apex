## Oracle APEX 24.2 install on Oracle Linux 8.9:




### Prerequisites:
- Oracle Linux Server 8.9
- Oracle Database 19c or 21c installed and running
- At least 2GB RAM, 10GB disk space
- Root or sudo access
- Required components
    - SQL*Plus or SQLcl
    - Java-11 or 17 (for ORDS later)

### Install Apex:

- Temp tablespace → usually `TEMP` (default)
- You don’t actually need `APEX_TEMP` unless you want custom setup.


_Parameters breakdown:_

| Parameter | Value | Meaning | 
| --------- | ----- | ------- |
| 1st | APEX | is the name of the tablespace for the APEX application user  | 
| 2nd | APEX | is the name of the tablespace for the APEX files user. |
| 3rd | TEMP | is the name of the temporary tablespace or tablespace group. |
| 4th | /i/ | is the virtual directory for APEX images. To support future APEX upgrades, define the virtual image directory as `/i/`. |

### Configure Nginx Laodbalancer: 



- `least_conn;` = “send to least busy server”
    - Sends new request to the server with the fewest active connections
    - Load is dynamically balanced based on current traffic (**recommended for load balancing**)

- `ip_hash;` = “same user, same server”
    - Same user → same server (session “stickiness”)
    - Good for session-based apps without shared session storage




```conf 

## cat nginx.conf
## Host based Nginx Reverse-Proxy & Load balancer

events {
    #empty
}

http {

    log_format main '$remote_addr - $remote_user [$time_local] '
        '$server_name to: $upstream_addr [$request] '
        'upstream_response_time $upstream_response_time '
        'msec $msec request_time $request_time ';

    upstream weblogic_cluster {
        least_conn;

        server 10.1.1.11:8080;  ## Tomcat Server IP & Port
    }

    server {
        listen 80;
        server_name _;

        access_log  /var/log/nginx/access.log  main;

        location /ords/ {
            proxy_pass http://weblogic_cluster;

            proxy_http_version 1.1;

            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;

            proxy_buffering off;
            proxy_request_buffering off;

            proxy_connect_timeout 60;
            proxy_read_timeout 60;
        }

        # Optional: static APEX images
        location /i/ {
        proxy_pass http://weblogic_cluster;

        proxy_set_header Host $host;
        }
    }

}
```



Or,


_Full Configuration:_
```conf 

## cat nginx.conf
## Path-based routing:

events {
    #empty
}

http {


    upstream weblogic_cluster {
        least_conn;

        server 10.1.1.11:8080;      ## Tomcat Server IP & Port

        keepalive 32;
    }


    server {
        listen 80;
        server_name _;


        # =========================
        # ORDS Application
        # =========================
        location /ords/ {
            proxy_pass http://weblogic_cluster/ords/;

            proxy_http_version 1.1;

            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;

            proxy_connect_timeout 60s;
            proxy_send_timeout 300s;
            proxy_read_timeout 300s;

            proxy_buffering off;
            proxy_request_buffering off;
        }

        # =========================
        # APEX static images (/i/)
        # =========================
        location /i/ {
            proxy_pass http://weblogic_cluster/i/;

            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;

        }

    }

}
```
